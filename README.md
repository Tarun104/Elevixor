# Elevixor

Elevixor is an AI-powered career growth platform designed to help students, freshers, and professionals build their careers with confidence.

## Features

- ATS Resume Building
- LinkedIn Profile Optimization
- Portfolio Website Development
- Interview Preparation Services
- Career Starter Guidance
- Data Analytics Services
- Software Development Solutions
- AI-Powered Career Assistance

## Tech Stack

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Node.js
- Express.js
- MongoDB
